WEB DEVELOPMENT ASSIGNMENT 1

  • Headings

    1. HTML headings are used to structure content on a webpage, making it easy for users and search engines to understand the organization and importance of information.

    2. Use headings in order, without skipping levels (e.g., do not jump from h1 to h3)

    3. Headings should describe the content that follows and help organize the page for both readability and accessibility

    4. Do not use headings just to enlarge or bold text; use them for structure and meaning

  • lists

    1. ORDERED LIST - Displays items in a specific sequence, such as steps or rankings. Each item appears with a number or letter by default.

    2. UNORDERED LIST - Presents items without any order, typically marked by bullets (- , ○, ■). This list suits things like groceries or features

    3. DESCRIPTION LIST - Used to define terms and their descriptions, like glossaries

    4. You can nest lists for hierarchical information (e.g., sub-lists within main lists), and mix ordered and unordered lists.

  • bold

    1. In HTML, text is made bold by using either the <b> tag or the <strong> tag, both of which display text in a bold style in browsers.

    2. <b> wraps content for visual emphasis, without semantic meaning or importance

    3. <strong> indicates that content is important in addition to displaying it bold, which helps with accessibility and SEO

    4. Prefer <strong> when the text is important or needs emphasis for screen readers and SEO.
